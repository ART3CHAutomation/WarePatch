Industria 4.0

Gracias a la arquitectura de red desarrollada en el proyecto, se logró el desarrollo remoto del sistema SCADA, como también su visualización remota. Con el fin de aplicar las nuevas tecnologías IoT, se utilizó el protocolo MQTT y una integración completa entre lsas diferentes versiones de Ignition.

Comunicaciones

MQTT

Para la conexión entre Ignition local e Ignition Cloud

OPC

Para la conexión entre dispositivos e Ignition

Node-Red

Para la conexión de dispositivos IoT con la nube mediante MQTT
