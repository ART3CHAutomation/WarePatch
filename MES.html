Sepasoft para una integracion con MES Rapida y Eficiente

Los software MES (Manufactory Executing System) o MOM (manufacturing operation management) son una capa de  funcionalidades que suelen conectar el ERP que es claramente software gestión empresarial y el SCADA, que es claramente gestión de planta por lo que sus fronteras son difusas. sin embargo se han encontrado 4 funcionalidades core dentro de todo software MES haciendo énfasis en las últimas 2 pues en algunos casos las primeras se relegan al ERP una vez  este es implementados 


  1. WORK ORDER GENERATION

  2. SCHEDWALLING 

  3. OEE tracking

  4. DOWNTIME TRAKING

A estas pueden agregarse algunas funciones bonus por así llamarlas como lo podrían ser

  • RECEPIE MANAGEMENT
  • DIGITAL QUALITY INSPECTION PLAS
  • STATISTICAL PROCESS CONTROL
  • ISO 9009 COMPLIANCE
  • DEVIATION TRAKING
  • CONSUMING WIP
  • CREATIVE WHIP

Sepasoft


Sepasoft comenzó en 2010.Como un socio estratégico de inductive automation, por lo que básicamente desarrollan los módulos MES para la plataforma Ignition, que a se usa para la coneccion entrenos diferentes servidores (robotstudio, nx, studio 500)
 su implementación en módulos permite la implementación y personalización de las funcionalidades que se ajusten  mejor a cada caso, así mismo es mucho más económico pagar por las funcionalidades que se usarán, un paquete integrado podria ser mas costoso 
se puede notificar a través del sistema de alarmas los problemas detectados por el sistema MES (un OEE Muy bajo)
Hay bastante información  para la implementación del sistema en  plataformas abiertas como youtube y se puede solicitar una versión demo directamente con la empresa.
